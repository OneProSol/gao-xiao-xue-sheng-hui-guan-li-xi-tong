# 高校学生会管理系统

一个基于Vue3 + SpringBoot + MyBatisPlus + MySQL的高校学生会管理系统。

### 一、相关技术
- 后端：`jdk`、`SpringBoot 2.x`、`MyBatisPlus 3.x`、`JWT`。
- 前端：`node`、`Vue 3`、`Vue Router`、`Element Plus`、`Axios`。
- 数据库：`mysql`。

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk17 或 jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm` 。
- 数据库：`mysql5.7以上`。

### 四、功能描述及功能图展示

系统分为：学生、学生会成员、管理员。

学生功能：1.登录、2.个人信息、3.活动参与、4.招新报名

学生会成员功能：1.登录、2.个人信息、3.活动参与、4.通知查看、5.积分查询

管理员功能：1.登录、2.成员管理、3.活动管理、4.通知管理、5.招新管理、6.积分管理、7.操作日志、8.角色管理、9.权限管理、10.部门管理
